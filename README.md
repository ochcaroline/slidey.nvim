# slidey.nvim

Neovim plugin for [Slidey](https://github.com/ochcaroline/slidey) - a terminal presentation tool.

## Why?

This is just to run current file (current buffer) in Slidey.

Checks:

- if Slidey is installed and in PATH
- if file is readable
- if file is Markdown

## How?

Enter the file you want to present (Markdown file) and press <leader>ss
